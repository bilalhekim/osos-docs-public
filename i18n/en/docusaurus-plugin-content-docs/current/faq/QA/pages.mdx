<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>What tools are available for customizing page designs?</b></summary>
<div>
There are many components that can be used to customize page designs, including: Title Component, Text Component, Media Component, Statistics Component, view Component, and more.
For more details on page elements, refer to the platform's page guide <a href="../../guide/information-structures-concepts/basic-concepts/pages">here</a>.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Are there options to add buttons and links for navigation between pages or sections?</b></summary>
<div>
Yes, buttons and links can be added for navigation. Please refer to the pages section in the platform usage guide <a href="../../guide/information-structures-concepts/basic-concepts/pages">here</a>.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can pages or interfaces be protected from unauthorized access?</b></summary>
<div>
Yes, this can be done by setting access permissions.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can multiple types of pages be created, such as a homepage and product or service pages?</b></summary>
<div>
Yes, many pages can be created, and the content of each page can be chosen individually.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can titles be added to elements, and can they be shown or hidden?</b></summary>
<div>
Yes, a title can be added to an element, and you can control its visibility.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can elements be shown or hidden based on the user?</b></summary>
<div>
Yes, the admin can customize the visibility of elements for targeted user segments through the custom display settings in the element's settings.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>What is the maximum number of pages or interfaces that can be created on Asas?</b></summary>
<div>
There is no limit to the number of pages that can be created.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can the page be formatted into columns and rows?</b></summary>
<div>
Yes, it can be divided into columns and rows using the paragraph and column elements.
</div>
</details>
