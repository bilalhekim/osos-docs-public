<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> What is the Platform? </b></summary>
<div>
It is a no-code platform used to create websites and mobile applications easily without the need to write code. It relies on drag-and-drop features to build and customize applications.
</div>
</details>


<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> What are the main uses of this platform? </b></summary>
<div>
The platform  is used to create various systems and services such as task management systems, project management systems, HR systems, and many other important applications.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> Can I create an app or website using the platform without prior programming knowledge? </b></summary>
<div>
Yes, anyone can create an app or website using the platform without prior programming knowledge using drag-and-drop interfaces. For more information, see: <a href="docs/guide/my-first-application">Building Your First Application</a>
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> Can I customize the design of the app or website according to my needs? </b></summary>
<div>
Yes, you can customize the page layouts according to your needs using the available tools. For more information, see: <a href="../../guide/information-structures-concepts/basic-concepts/pages"> Pages </a>
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> Can I add custom CSS styles? </b></summary>
<div>
Yes, you can do this by modifying the Additional Styles section in the page details. For more information, see: <a href="../../guide/information-structures-concepts/basic-concepts/pages"> Page Management </a>
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> What are drag-and-drop tools and how can they be used? </b></summary>
<div>
Drag-and-drop tools are an interactive interface that allows users to drag specific elements and drop them in other places within the site visually and simply, instead of writing code. They can be used, for example, to build application pages.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> Is the platform completely no-code or low-code? </b></summary>
<div>
The platform is 97% no-code, but JavaScript expressions can be added in some specific cases.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> Can the platform be used by the same user on multiple devices? </b></summary>
<div>
Yes, the platform can be used with the same account on multiple devices and in multiple sessions.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> Are there notifications or alerts to help users track their activities on the platform? </b></summary>
<div>
Yes, there are 3 types of notifications on the platform (System Notifications - Email - SMS).
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b> Are there any restrictions on the number of applications or services the platform can integrate with? </b></summary>
<div>
There are no restrictions on the number of applications and services that can be integrated as long as the integration requirements are met.
</div>
</details>
