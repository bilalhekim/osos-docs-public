<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>How can I create databases on the platform?</b></summary>
<div>
There is no concept of a database in the platform, but there is the concept of modules and entities which serve the function of databases and their associated tables. To learn how to create <a href="../../guide/information-structures-concepts/basic-concepts/modules">modules</a> and <a href="../../guide/information-structures-concepts/basic-concepts/entities">entities</a>, check the provided links.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>What types of data can be stored in the platform?</b></summary>
<div>
It supports a variety of fields that include the following data types: text data, numeric data, multimedia data, and composite data sets such as files and maps.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can custom data fields be added?</b></summary>
<div>
Yes, custom data fields can be added using the drag-and-drop function. There are many types of custom fields available. For more details about the fields on the platform, check <a href="/docs/guide/information-structures-concepts/basic-concepts/fields">Fields</a>.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can data be deleted on the platform?</b></summary>
<div>
No, data cannot be deleted but can only be archived.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can relationships be created between entities on the platform?</b></summary>
<div>
Yes, relationships can be created between entities. For more information, see <a href="../../guide/information-structures-concepts/basic-concepts/relations">Relationships</a>.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can data be filtered?</b></summary>
<div>
Yes, data can be filtered using the filter icon if the administrator has allowed it.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can data be exported?</b></summary>
<div>
Yes, data can be exported to formats such as Excel or CSV.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can data be imported from other sources?</b></summary>
<div>
Yes, data can be imported from other sources through integration.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Are there limits to the amount of data or the number of records that can be stored?</b></summary>
<div>
There are no limits to the amount of data or the number of records on the platform, but some field types may have limits in size and number.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can data be protected and access permissions controlled?</b></summary>
<div>
Yes, data can be protected and access permissions controlled through the security and permissions settings available on the platform.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can platform data be integrated with other external applications?</b></summary>
<div>
Yes, data can be integrated with other external applications.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>What tools are available for data analysis and statistics extraction?</b></summary>
<div>
Various statistics can be generated based on queries and conditions in the dashboard and reports.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can data be exported to formats like Excel or CSV?</b></summary>

</details>
