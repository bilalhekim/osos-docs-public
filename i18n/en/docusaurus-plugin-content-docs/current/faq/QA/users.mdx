<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can an account be created on the platform?</b></summary>
<div>
No, an account cannot be created by users. Only the system administrator can create accounts. For more details on adding users, please refer to the users guide on the platform <a href="../../guide/information-structures-concepts/basic-concepts/users">here</a>.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>How can I know that a new account has been created for a user? How can the password be set?</b></summary>
<div>
When an account is created, a notification is sent to the email, indicating that the account has been created and includes a link through which the user can set their password.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>What information does the administrator need to create an account on the platform?</b></summary>
<div>
The administrator needs the user's first name, last name, and email address to create the account. Other personal information like birth date, gender, phone number, and profile picture can be provided but is not mandatory.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>What permissions are granted to the user when an account is created?</b></summary>
<div>
By default, the account has no permissions. The administrator can assign permissions to the account later based on the user's role.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>How can users change their password if they forget it?</b></summary>
<div>
On the platform's login page, there is a "Forgot Password" option where an email is sent to the user to reset their password.
</div>
</details>

<details style={{background: 'none', border: 'none', boxShadow: 'none'}}>
<summary><b>Can users change their profile picture?</b></summary>
<div>
Yes, users can change their profile picture through their profile page.
</div>
</details>
